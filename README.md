# AI Meeting Copilot

AI-powered meeting assistant for real-time transcription, summarization, and action item extraction.

## About
AI Meeting Copilot enhances meeting productivity by providing live transcription, automatic summarization, and intelligent extraction of action items and decisions. Designed to integrate with common video conferencing platforms.

## Tech Stack
- Language: Python
- Speech: Real-time transcription pipeline
- NLP: Summarization and entity extraction
- Integration: Video conferencing platform hooks

## Status
Active development - transcription engine and summarization pipeline established.
